# UIの変更

## 今回用いるUIの変更方法(CSS)

### CSSで用いられる単語の意味

padding：内側の余白。箱の中の隙間を管理する。　(ex. padding: '15px 20px' 縦 横の順で書く)

margin：外側の余白。他の要素との距離を管理する。

marginTop, marginBottom：上だけ、下だけの余白を生成

margin: '0 auto'；要素を左右中央に寄せる魔法の合言葉。?

maxWidth: 最大の横幅。これ以上広がらないように制限する。?

minHeight：最小の高さ。100vh は「画面の高さ100%分」という意味。?

width, height：幅と高さ。

backgroundColor：背景の色。

color：文字の色。

border：枠線。1px solid #ddd は「1pxの太さ・一本線・薄いグレー」という意味。

border: 'none'：枠線を消す。

borderRadius：角の丸み。数値が大きいほど丸くなる。

boxShadow：影。

opacity：透明度。0.7 は 70% の濃さ。

fontSize：文字の大きさ。

fontWeight：文字の太さ。

textAlign：文字の揃え方。center で中央、left で左寄せ。

fontFamily：フォントの種類(書体)。

letterSpacing：文字と文字の間の距離。

display: 'flex'：横に並べるという宣言。

flexDirection：'column': flexを使っているけど「縦に並べる」時に使う。？

justifyContent：並べた時の左右の配置。

    center: 中央寄せ。

    space-between: 両端に広げて配置。

alignItems：並べた時の上下の配置。

    center：上下の中央で揃える。

gap：並べた要素同士の「あいだの隙間」。
